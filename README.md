# Dream Duas — Visionaire App

A guided, interactive workbook app to help Muslims of all ages build their **6–7 Dream Duas** using the Visionaire methodology by Shaykh Muhammad al-Shareef.

## ✨ Live App
👉 [Open the App](https://haleema1.github.io/dream-duas/)

## 📖 About

This app guides users step-by-step through building powerful, specific, Akhira-connected duas across all areas of life — health, wealth, deen, relationships, and more.

**Five versions available:**
- ⭐ **Kids** (Ages 9–11) — fun, emoji-rich, Pokémon-evolution framing
- ✨ **Teen Girls** (Ages 14–17) — identity, confidence, hijab, inner beauty
- ⚡ **Teen Boys** (Ages 14–17) — character, brotherhood, gaze, strength
- 💚 **Adult Mom** — self-care, children's legacy, marriage, joy
- 🏔️ **Adult Dad** — leadership, legacy, spiritual fatherhood, purpose

## 🔒 Privacy

All data is saved locally in the browser (`localStorage`). Nothing is sent to any server. No account required.

## 🚀 How to Use

1. Open `index.html` in any browser — no internet needed
2. Select your version
3. Enter your name (optional)
4. Work through the foundation and 6–7 category screens
5. View and print your complete Dream Duas summary
